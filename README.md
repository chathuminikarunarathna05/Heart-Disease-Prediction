# Heart Disease Prediction using Machine Learning

## 📌 Project Overview
This project predicts heart disease using a Random Forest Classifier.

## 📊 Dataset
- Heart Disease Dataset (CSV)
- Contains medical and lifestyle attributes

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🚀 Model Used
- Random Forest Classifier

## 📈 Results
- Achieved high accuracy in prediction
- Important features identified: Age, Cholesterol, Blood Pressure

## 📂 Files Included
- Heart Disease Prediction using ML.ipynb
- heart_disease.csv
